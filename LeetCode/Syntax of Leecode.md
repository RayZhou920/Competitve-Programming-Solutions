1.Java .sqrt() method with Examples
======
````Java
import java.lang.Math;
class Gfg {
    // driver code
    public static void main(String args[])
    {
        double a = 30;
  
        System.out.println(Math.sqrt(a));
  
        a = 45;
  
        System.out.println(Math.sqrt(a));
  
        a = 60;
  
        System.out.println(Math.sqrt(a));
  
        a = 90;
  
        System.out.println(Math.sqrt(a));
    }
}

Output:
5.477225575051661
6.708203932499369
7.745966692414834
9.486832980505138
````
